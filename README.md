# MUTAMBUKE

A real-time ride-hailing platform for motorcycles (Moto) and taxis in Rwanda.

## Features
- **Real-time Tracking**: Live GPS updates for riders and passengers.
- **Multi-language Support**: Kinyarwanda, English, and French.
- **Dual Dashboard**: Dedicated experiences for Passengers and Riders.
- **Admin Management**: Verification system for new riders.
